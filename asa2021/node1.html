Overview

  • A frequency-domain method to estimate 1D input impedance for acoustics systems that can be approximated by piece-wise cylindrical and conical segments (with extensions for side holes and more arbitrary sub-structures).

  • Sometimes referred to as the “transmission matrix method.”

  • Generally used for wind instruments (air columns) but it can be applied to other 1D systems such as strings (though perhaps less relevant because strings are normally continuous between terminations).

  • Also commonly used in optics for electromagnetic waves but I will focus on acoustics.

  • Input impedance can be used to evaluate mode frequencies (relative strengths, harmonicity) that can in turn be used to estimate ease of play, intonation,

  • Can be used as a comparison when making measurements (reality check, assess model quality).

  • Can be used for sound synthesis, either by devising a modal filter representation or by computing time-domain impulse responses or reflection functions using the IFFT.

  • Most listeners probably already know the method well. The focus here is on a toolbox of scripts that I am sharing to simplify the TMM process (and which might be useful for teaching and student research).
